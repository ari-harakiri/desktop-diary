# DesktopDiary

This repository contains the public GitHub Pages version of DesktopDiary.

## Normal website updates

The website's code and styling are compiled into one file:

- `desktop-diary.html`

For a normal code or design update, replace that file in GitHub Desktop, review
the change, write a short summary, and click **Commit to main** followed by
**Push origin**.

## Artwork folders

The `assets` and `companions` folders contain images, fonts, sprite sheets, and
the Sudoku game. Leave these folders in place. They only need to be updated when
artwork, a font, or the Sudoku game changes.

## Important

- `index.html` sends visitors to `desktop-diary.html`.
- Keep the filename `desktop-diary.html` unchanged.
- Do not upload the development-only `scripts` or `styles` source folders here.
- Saved diary data is stored by the browser and cloud services; this repository
  does not contain anyone's saved diary entries.
